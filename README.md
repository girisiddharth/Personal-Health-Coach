

A next-generation health monitoring agent with personalized recommendations, powered by Gemini and ScaleDown technology.

## Features
- **Dashboard**: Real-time health metrics (Calories, Steps, Sleep, HR).
- **AI Coach**: Personalized advice using Gemini Pro, context-aware of your health data.
- **ScaleDown Integration**: Simulates 80% data compression for efficient long-term context analysis.
- **Wearable Hub**: Connect Fitbit and Apple Health (Simulated).

## How to Run
1. Open the `index.html` file in any modern web browser (Edge, Chrome, Firefox).
2. No installation required (Zero-dependency architecture).

## Setup
To use the **AI Health Coach**:
1. Get a free Gemini API Key from [Google AI Studio](https://aistudio.google.com/).
2. Go to the **Integrations** tab in the app.
3. Paste your API Key and click **Save**.
4. Navigate to **AI Coach** and start chatting!

## Technologies
- Vanilla HTML5/CSS3 (Glassmorphism Design)
- Modern JavaScript (ES6+)
- Google Gemini API (REST Integration)
